소개

jellyTranslator는 번역을 돕는 크롬브라우저의 확장프로그램(extension) 입니다

jellyTranslator는 내부적으로 translate.google.com을 사용합니다.

현재 popup_translate과 page_translate기능 2가지를 지원합니다.

팝업 번역

페이지 번역(Beta)

git 파일 로딩때문에 최초 재생시 느려 보입니다.

페이지 번역은 아직 beta버전입니다.

오류가 발생하면 밑의 오류해결 항목에서 해결해보세요.

단축키 들

// ShortCuts Setting
"pop_up" : {
  "suggested key": {
    "default" : "Ctrl+Shift+X",
    "mac" : "Command+Shift+Z",
    "linux" : "Ctrl + Shift + X",
  }
}

// ShortCuts Setting(coming soon...)
"translate_page" : {
  "suggested key": {
    "default" : "not supported",
    "mac" : "not supported",
    "linux" : "not supported",
  }
}

Bugs

page_translate bugs

  1. 페이지 번역중 멈추는 경우

    2. 너무 많은양의 번역을 하면 멈출수있습니다.

    popup_translate을 띄우고 아무거나 입력하시면 새창에 recapcha가 생성됩니다.

    사람임을 증명하고 다시 사용하시면 됩니다.

popup_translate bugs

  • 명령어가 가끔씩 안먹히고 번역되는 버그

    • 마지막문자 지우고 다시 천천히 입력하세요

  • 특수문자가 들어간경우 발번역되는 현상

    • 번역은 전부 머신러닝으로 학습된 translate.google이 해줍니다.

    근데 걔네도 이상한기호 막나오면 발번역합니다. 적당히 기호를 없애서 돌리세요
